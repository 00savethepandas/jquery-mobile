Listview: Reveal

Home Search

With the reveal extension, you can allow a list item to reveal content on swipe or hover/click by adding a data-reveal="true" to the list item and placing revealable content inside that list item in an element with data-reveal-disclose="true"

With this in place, touch users will be able to swipe a list item to reveal content underneath it. Mouse users will see a portion of the row reveal on hover, and can click the revealed portion to see more.

For example:


<li data-reveal="true"><a href="index.html">
  <h3>Stephen Weber</h3>
  <p><strong>You've been invited to a meeting at Filament Group in Boston, MA</strong></p>

  <form data-reveal-disclose="true" action="#">
    <h3>Delete item?</h3>
    <input type="submit" data-iconpos="notext" data-icon="delete" data-theme="c" data-shadow="false">
  </form>
</a></li>

Note that the type of element that has the data-reveal-disclose="true" attribute does not matter.
