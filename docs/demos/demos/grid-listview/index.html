Grid Listview

A regular listview on smartphones, but grid tiles on tablets and larger screens? This demo shows you how you can accomplish this with custom CSS.

Starting points

Create a listview from an unordered list. For this demo we use an inset listview to show you how you can apply the corner styling to the tiles as well.

We will use two breakpoints. At the first breakpoint we swap from a stacked layout to a three column grid layout. At the second we swith from three to four columns.
