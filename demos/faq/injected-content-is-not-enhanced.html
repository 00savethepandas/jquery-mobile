Question:

Content injected into a page is not enhanced.

Answer:

jQuery Mobile has no way to know when you have injected content into a page. To let jQuery Mobile know you have injected content that must be enhanced you need to either make sure the plugins are called to enhance the new elements or trigger("create") on the container so you don't have to call each plugin manually


//HTML 
<form id="formid">
	<input type="search" id="searchInput"/>
	<button id="submitButton">Submit</button>
</form>
//javaScript **CORRECT**
$("#formid").trigger("create");
//javaScript INCORRECT
$("#serchInput").trigger("create");
$("#submitButton").trigger("create");

An important not to remember and common mistake it to make sure you trigger create on the parent container and not on the actual element which needs to be enhanced.
